# 工业富联（601138）研究笔记

按时间倒序，记录讨论中产生的认知共识。

## 2026-03-11 公司治理结构与少数股东地位

**状态**：✅ 已验证

**核心认知**：A股投资者持有的是一个毫无话语权的控股壳公司少数股东权益。

**推理过程**：

1. FII（上海上市主体）本身是纯控股壳——固定资产仅800万，无独立经营能力，640亿长期股权投资就是全部"资产"。

2. 控制链层层空壳：鸿海精密（台湾，无实际控制人）→ 中坚企业（香港空壳）→ 7-8个境外壳公司分散持股 → FII → 萨摩亚控股公司 → CNT-SG/墨西哥工厂/中国工厂。每一层都不是真正的经营实体。

3. 鸿海体系掌握实际运营控制权：工厂管理、客户关系、供应链、人员调配走鸿海全球体系。FII从鸿海体系采购236.6亿、租赁厂房设备11.4亿、支付商标使用费——生产要素都不在FII手中。

4. 84.1%投票权意味着关联交易审批走程序。鸿海可通过转移定价调节FII体内利润。外部投资者无法验证关联交易定价的公允性。

5. 利润→现金→分红的漏斗极窄：353亿净利润 → 52亿经营现金流（OCF/NI=0.15）→ 65.5亿拟分红（派息率18.6%）。大部分利润以美元形式留在海外体系内循环。

**结论**：这个结构天然造成上行分享有限（鸿海拿走84%好处）、下行风险放大（鸿海可转移利润，少数股东承受更大比例损失）的不对称。理应有显著的治理折价（参考港股分拆子公司20-40%、韩国财阀30-50%），但A股市场给了PE 30x / 5年88%分位的溢价——因为这是少数能直接参与AI算力叙事的A股标的。

## 2026-03-11 业务独占性来源

**状态**：✅ 已验证

目前鸿海体系内没有其他子公司从事AI服务器业务，FII具有事实上的独占性。但这个独占性来自鸿海的业务分配安排，而非FII自身的独立能力——业务边界由鸿海划定。这反而是FII最大的"护城河"：不来自技术或客户，来自体系内分工格局。格局一旦变化，估值基础动摇。

## 2026-03-11 母子公司市值倒挂

**状态**：✅ 已验证

鸿海精密（2317.TW）市值约$1,083亿，FII市值约$1,470亿。鸿海持有FII 84.1%，对应约$1,236亿——已超过鸿海自身全部市值。鸿海除FII外所有业务（iPhone代工、电动车、Sharp、半导体等）隐含估值为负。这是A股对AI叙事溢价的极端体现，也说明FII当前定价缺乏独立基础。**研究FII时应同步关注鸿海（2317.TW）股价作为估值锚。**

## 2026-03-11 美国子公司与资产可移动性

**状态**：✅ 已验证

FII名下有9个美国实体，最大的Foxconn Assembly LLC（注册资本$2.14亿，服务器组装）和新设的Fii Holdings USA Inc.（$3.45亿，电子制造+金融规划）。这些实体大部分通过"同一控制下企业合并"从鸿海体系划入，不是FII独立建设的。

关键风险：
- **双向监管尴尬**：美国CFIUS角度，这是台湾公司通过中国A股上市公司间接持有的美国服务器工厂；中国角度，A股上市公司的核心资产（71.7%在境外）和利润大量在海外。两边都可能产生审查压力。
- **资产可移动性**：鸿海系极其复杂（消费电子/iPhone、电动车Foxtron、半导体、Sharp、数字健康、机器人等多条业务线），资产在体系内实体间的划拨是常规操作。FII名下的资产组合不是固定的——鸿海可以根据整体利益随时调整边界（装入拉估值，移出避风险），少数股东无权干预。投资者买的不是一组确定资产，而是鸿海随时可能重新定义的"业务窗口"。

## 2026-03-12 鸿海视角：FII的集团定位与业务演变

**状态**：✅ 已验证

**鸿海对FII的定位**：标普评级指出FII毛利润和净利润占鸿海的比重分别在四成以上和六成以上，是鸿海主要利润和现金流贡献者，鸿海对FII的外部支持评估为"极高"。但这个利润贡献比例本身就是鸿海选择的结果——转移定价、关联采购比例（已从2023年48%升至2025年58%）、商标使用费、厂房租赁费，每一项都是调节旋钮。中美监管压力更给了鸿海调整的正当理由：美国CFIUS可能收紧对中国关联实体的审查（推动产能移出FII体系），中国监管可能关注境外资产和利润外流（推动压缩FII境外业务边界）。两个方向都会削弱FII的利润贡献，而鸿海可以将其包装为"合规调整"。所以"FII贡献六成利润"不是护城河，是当前快照——随时可变。

**IPO时的业务构成（2018年）**：通信及移动网络设备占营收62.76%（毛利率11.1%），云计算（含服务器）仅占37.11%（毛利率4.2%）。IPO叙事核心是"工业互联网第一股"，募资271亿聚焦工业互联网平台。服务器在当时是低毛利的配角。

**工业互联网转型失败**：上市后推出工业云平台（Fii Cloud）、专业云（Micro Cloud）等，但工业互联网业务从未成功对外商业化，案例局限于集团内部。这进一步印证"护城河来自鸿海体系内分工而非自身能力"的判断。

**资产组合的被动演变**：
- 墨西哥工厂2009年建成，最初做消费电子组装（北美最大）
- AI服务器产能是2024-2025年才紧急扩建的（2024年2月投$3000万购地，5月再投$1.2亿，与英伟达合作AI工厂2025年初投产）
- 资产组合不是FII主动规划的战略布局，而是被动跟随客户需求（北美云厂商就近部署）和鸿海集团安排

**鸿海集团战略框架**：
- 集团层面"3+3"：三大新兴产业（电动车、数字健康、机器人）+ 三大核心技术（AI、半导体、新一代通信）
- FII层面"2+2"：核心业务（云计算+通信网络设备）+ 延伸（半导体封测、新能源汽车零部件）+ 新事业（大数据、机器人）
- FII的"2+2"与鸿海"3+3"高度重叠（电动车零部件、机器人），说明FII的战略方向由集团顶层设计驱动，而非独立制定

## 2026-03-12 双重换汇结构与低派息的结构性解释

**状态**：✅ 已验证

FII的分红路径存在双重换汇：运营实体的美元利润先结汇为人民币（FII是A股主体，必须以人民币派息），鸿海的境外持股实体收到人民币分红后还要换回美元、经中国资本项目外汇审批汇出。对鸿海而言，通过FII分红是最低效的取钱方式，远不如在运营实体层面通过关联交易（采购236.6亿、租赁11.4亿、商标费等）直接在海外体系内截留利润，完全绕开换汇和审批。

这解释了为什么派息率只有18.6%：低派息不只是鸿海不愿意分，而是结构本身让分红成为最不合理的利润分配路径。鸿海的理性选择就是少分红、多走关联交易。A股少数股东的利益在结构设计层面就被边缘化了。

## 2026-03-12 增长加速监管风险暴露

**状态**：✅ 已验证

AI服务器增长不是稀释治理风险，而是催化监管风险——速度越快，暴露越早。

推理链：
1. AI服务器增长越快 → FII境外资产和利润占比越高 → 中国监管对境外资产和利润外流的关注加剧
2. AI服务器增长越快 → FII在美国AI基础设施中的角色越大 → CFIUS审查压力加剧
3. AI服务器增长越快 → 关联交易规模越大 → 转移定价的灰色空间越大，外部投资者越难验证
4. FII越具战略敏感性 → 越可能被中美任一方要求"合规调整" → 鸿海获得正当理由重新划定业务边界

结论："AI增长能稀释治理风险"这条看似最有力的反论，实际上是治理风险的加速器。找不到能实质性反驳治理结构问题的论据——治理折价的逻辑链非常坚固。
