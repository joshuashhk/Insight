## 核心标准（每次分析都适用）

### 数据源优先级

1. 10-K / 年度报告
2. 财报电话会 / 投资者演示
3. 卖方研究
4. 行业报告（麦肯锡、Gartner 等）
5. 新闻报道

### 数据可比性要求

- 时间区间必须一致
- 指标定义必须一致
- 货币统一换算
- 缺失数据用 "-"；私有公司数据用 "N/A" 或 "[E]"
- 每个数字都要标注来源："[公司] [文件] ([日期])"

## 分析工作流（9步）

### Step 0：确定行业定义指标

| 行业 | 核心指标 |
|------|---------|
| SaaS | ARR、NRR、CAC 回收期、LTV/CAC、Rule of 40 |
| 支付 | GPV、take rate、attach rate、交易利润率 |
| 市场平台 | GMV、take rate、买卖方比例、复购率 |
| 零售 | 同店销售、库存周转、坪效 |
| 物流 | 量、单位成本、准时率、产能利用率 |

### Step 1：市场背景

- 市场规模（附来源）；增长驱动与阻力；关键趋势

### Step 2：行业经济结构

- 价值流向图：垂直整合型 / 平台网络型 / 分散型

### Step 3：目标公司画像

```
| 指标     | 数值         |
|---------|-------------|
| 营收     | $4.96B      |
| 增速     | +26% YoY   |
| 毛利率   | 45%         |
| 盈利能力 | $373M EBITDA |
| 客户数   | 134K        |
| 留存率   | 92%         |
| 市占率   | ~15%        |
```

多业务分部公司，补充各分部明细（营收、增速、占比、EBITDA、利润率）。

### Step 4：竞争对手分组

按以下维度分组：商业模式 / 细分市场 / 竞争姿态 / 起源背景

### Step 5：定位可视化

| 可视化方式 | 适用场景 |
|-----------|---------|
| 2×2 矩阵 | 两个主导竞争因素 |
| 雷达图 | 多维度综合比较 |
| 层级图 | 自然聚类 / 战略分组 |
| 价值链图 | 垂直行业 |
| 生态系统图 | 平台市场 |

### Step 6：竞争对手深度解析

- 表一（量化）：营收、增速、毛利率、市值、盈利能力、客户数、留存率、市占率
- 表二（定性）：业务描述、优势、劣势、战略

### Step 7：横向比较

```
| 维度   | A 公司       | B 公司      | C 公司      |
|--------|-------------|-------------|-------------|
| 规模   | ●●● $160B   | ●●○ $45B   | ●○○ $8B    |
| 增速   | ●●○ +26%   | ●●● +35%  | ●●○ +22%  |
| 利润率 | ●●○ 7.5%   | ●○○ 3.2%  | ●●● 15%   |
```

评级后面跟实际数字（"●●● $160B"，不是只有"●●●"）

### Step 8：战略背景

- 并购交易、合作关系、融资、监管进展

### Step 9：综合研判

**护城河评估框架：**

| 护城河类型 | 评估内容 |
|-----------|---------|
| 网络效应 | 用户/供应商飞轮强度 |
| 转换成本 | 技术集成深度、锁定程度 |
| 规模经济 | 量大后的单位成本优势 |
| 无形资产 | 品牌、专有数据、牌照、专利 |

每类评：强 / 中 / 弱，附支撑证据。

**投资情景框架：**

```
| 情景 | 概率 | 关键驱动 |
|------|------|---------|
| 乐观 | 30% | 市占率提升、利润率扩张 |
| 基准 | 50% | 当前轨迹延续 |
| 悲观 | 20% | 竞争压力、利润率压缩 |
```

## 附录：2×2 矩阵常用轴对

**科技/SaaS：** 产品广度 × 客户分部；集成深度 × 地理覆盖

**消费/零售：** 价格定位 × 产品范围；线上 × 线下

**金融服务：** 产品复杂度 × 客户成熟度；规模 × 专业化

**医疗：** 诊疗场景 × 支付方结构；技术赋能 × 服务广度

**工业：** 定制化 × 规模；地理覆盖 × 垂直专注

## 写作原则

- **标题 = 洞察**（"规模领导者正在拉开差距"，而非"竞争分析"）
- **评级附实际数字**（"●●● $160B"，不是只有"●●●"）
- **先写提纲，审阅后再创作最终版**
